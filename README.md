# Briefeingangsregister
In diesem Repository werden Daten und cypher-Befehle zum Beitrag Andreas Kuczera, Das Nürnberger Briefeingangsregister als Quelle für die Informationsverarbeitung in der Stadt des 15. Jahrhunderts, in: Informationsverarbeitung in der Stadt des 12.-16. Jahrhundert. Beiträge des interdisziplinären (Post-)Doc-Workshop des Trierer Zentrums für Mediävistik im November 2018, hrsg. von Eric Burkart und Vincenz Schwab veröffentlicht.
